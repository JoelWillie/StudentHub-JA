# Student Life Manager - Mobile Web App
### Council of Community Colleges of Jamaica (CCCJ)
**Course**: Introduction to Mobile Application Development (CSYS2101)  
**Assessment**: Coursework I – Group Project (30%)  
**Semester**: September 2025 – Semester 1  
**Due Date**: November 26, 2025  

## Project Overview
This is a simple, mobile-friendly web application built with pure HTML, CSS, and JavaScript (no frameworks) that helps students manage their daily school life. The app contains four required pages:

1. **Home Page** – Welcome message + navigation menu  
2. **Class Schedule** – Weekly timetable displayed in HTML table  
3. **To-Do List** – Add/remove tasks with persistence using `localStorage`  
4. **About Page** – Information about the school, team members, and app purpose.
